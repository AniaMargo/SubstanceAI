Analytic Approach

To answer the first research question, we fit a linear model demonstrating the relation between parental monitoring and substance use using ordinary least squares (OLS) regression in R Version 2023.03.1+446 (R Core Team 2023). The packages used for analysis were Tidyverse (Wickham et al. 2019), here (Müller 2020), rio (Chan et al. 2018), ggplot2 (Wickham 2016), and janitor (Firke 2023).
